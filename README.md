# PiedPiper

## Описание проекта
Данный учебный проект представляет одностраничный сайт.
Проект создан в рамках прохождения курса от Stepik "Веб-разработка для начинающих: HTML и CSS".

## Стек технологий:
- HTML5;
- CSS3:
  - Flexbox;
  - Positioning;
- Методология БЭМ;
- Фйловая структура Nested БЭМ;
- Вёрстка по макету в Figma.
